# Step by step guide on how to setup local files from scratch

This is for me because I always forget how to do things. If you're not me, you might not want/need/be able to do some steps.

### 1. Installing Node.js

- Go to <https://nodejs.org/en/>
- Download the version that's recommended for most users.
- Install it.

### 2. Installing the typescript template

- Go to the [official typescript template github](https://github.com/bitburner-official/typescript-template)
- Click the green 'Code' button.
- Press the Download Zip button.
- Extract where you want and rename the folder to what you want.

### 3. Installing git

- Go to <https://git-scm.com/downloads> and install git bash.

### 4. Installing npm

- Open a terminal window.
  - type `d:` if the project directory is on the 'D' drive.
  - type `cd` + project files root directory.
  - type `npm install`.

### 5. Cloning this repository

- Go to the [repository](https://github.com/Darvass69/bitburner)
- Click the green 'Code' button.
  - In the 'HTTPS' tab, copy the URL.
- In Git Bash
  - Type `cd` + directory
  - Type `git clone` + URL
  - Put files in the desired folder (don't forget the .git hidden folder)
