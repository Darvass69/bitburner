# Steps to start the external editor with bitburner

These are the steps needed to have the external editor working.

### 1. Start the watch services

- `shift + alt + r` to open file with window's file explorer
- double click `startWatch.bat`

### 2. Go in bitburner

- Options -> Remote API -> type in the port: `12525` -> click connect. The icon should turn green and say it's online.
- Your firewall may yell at you again; allow the connection.
